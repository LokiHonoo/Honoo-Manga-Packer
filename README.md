## INTRODUCTION

漫画和图包的打包辅助工具。支持 ZIP/RAR/7Z/PDF 解压缩或重新打包到无压缩 ZIP 文件。

支持试用解包密码。此功能部分可用。

支持移除广告页。因功能所限软件仅有简单的添加广告页功能，未来考虑制作独立的广告页采集软件。
